1. What is responsible for handling async operations in Node.js?
A. V8 Engine
B. libuv
C. Call Stack
D. Garbage Collector

2. Which of the following runs on the main thread?
A. File system operations
B. Crypto hashing (async)
C. JavaScript execution (call stack)
D. Thread pool tasks
Ans: C

3. What happens when you use fs.readFileSync?
A. It goes to thread pool
B. It is non-blocking
C. It blocks the event loop
D. It runs in callback queue

4. Which API uses the thread pool internally?
A. setTimeout
B. Promise
C. fs.readFile
D. process.nextTick

5. Order of execution (simplified)?

B. Event Loop → Call Stack → Callback Queue

const crypto = require("crypto");
for (let i= 1; i<= 6; i++) {
    crypto.pbkdf2("pass", "salt", 1000000, 64, "sha512", () => {
        console.log(i);
    });
}
Assume default configuration.
Which statement is correct?
A. All 6 tasks run in parallel simultaneously
B. Only 1 runs at a time because Node is single-threaded
C. 4 tasks run in parallel, rest queued
D. Order of output is always 1 → 6
